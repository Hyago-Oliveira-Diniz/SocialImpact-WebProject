# 🌍 Projeto Desabrigados

Site desenvolvido com foco em conscientização social sobre pessoas em situação de rua no Brasil.

---

## 📌 Sobre o projeto

Este projeto foi criado com o objetivo de informar e sensibilizar usuários sobre a realidade de pessoas desabrigadas, abordando:

* Principais causas
* Notícias relacionadas
* Formas de ajudar

---

## 🚀 Funcionalidades

* Página inicial com apresentação do tema
* Página de causas da situação de rua
* Página de notícias
* Página com formas de ajudar
* Layout responsivo

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

---

## 📂 Estrutura do projeto

* `index.html` → Página principal
* `causas.html` → Explicação das causas
* `noticias.html` → Notícias relacionadas
* `ajudar.html` → Formas de ajudar
* `styles.css` → Estilização do site

---

## 🎯 Objetivo

Projeto acadêmico desenvolvido para prática de:

* Estruturação de páginas web
* Responsividade
* Organização de conteúdo
* Desenvolvimento com HTML e CSS
